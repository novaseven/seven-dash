# Seven Dash
A super simple <Dash /> component for react-native to draw customisable dashed lines

## Introduction

Seven Dash customized for self-use. 

## Installation
```
"dependencies": {
    "seven-dash": "git+https://git@github.com/novaseven/seven-dash.git#master"
    }
```
